I'm a coder.

Being a millenial, I've always been surrounded in my life by computers. They can be found built into so many things nowadays - even in places where one wouldn't even care to look. That's why I chose to study computer science - because it's so useful in so many fields and has so many applications - and that's also why I love to code things.

I'm fairly active on GitHub, a site for open-source projects to publish themselves and attract others to contribute code and patches. I'm a maintainer for the Homebrew-Cask package manager for the macOS operating system, and I submit patches to its sister/parent project as well, Homebrew proper.
